Update Charging Station Firmware 1. Ensure the device shows as Connected on acharger.ca. Ensure the charger is not connected to the vehicle. 2. On the Charging Stations tab select the Smart Charging Station. The outline box will appear red when the station is selected. 3. Select the Firmware Update button in the top menu. The Firmware Update box will open. 4. Select the Start Upload button. 5. Wait for Upload Stage 1 and Upload Stage 2 to complete. Do not interrupt Wi-Fi connection or disconnect the unitâ€™s power during the update. When the update is complete both stages will be at 100% and the charger indicator light will go back to Blue/Light Blue alternating.
