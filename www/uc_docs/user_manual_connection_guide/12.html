Charge and Get Report 1. Insert the charging Connector into the EV and ensure the connector is fully seated/locked in place. Once complete, the charging session will begin. 2. Ensure the Charging Status shows charging. 3. When the charging session is finished remove the charging connector and return to the holster. 4. Go to the Reports tab. 5. Select the charging station from the dropdown menu. 6. Select the date range for the reports. 7. Select Get Report. 8. Report will display Amperage (A) and Kilowatts (kW) over time (minutes).
