Charging Schedule Set Up Charging Schedule 1. Go to the Charging Profile tab. Ensure the correct Charger Serial number displays at the top of the screen. 2. Select the Add Profile button from the top menu. 3. Set the correct time zone and daylight saving setting. Select the Update button. 4. Set the Valid From and Valid To dates. 5. Select the date and time for the Start of the Schedule. The vehicle will start charging at the time selected. 6. Set the Charge Limit to the Maximum Current or Power. For example, if you want to charge at 32A for the first time period, type 32 into the Charge Limit field. 7. Set the Duration for the number of hours you want to charge at the Charge Limit. For example, if you wanted to charge at 32A for 2 hours, type 2 in the Duration field. 8. Select the Add New Period. The Duration (minutes) and Charging Current (A) will be graphed on the profile screen. 9. Add additional Charging Periods as desired. Set the Charge Limit and Duration for the next period. Amperage and Duration will be graphed on the profile screen.
